Software Gratuito Nunca Se Vio Tan Increíble

Blender es la suite de creación 3D gratuita y de código abierto. Soporta la totalidad del pipeline 3D: modelado, rigging, animación, simulación, renderizado, composición y seguimiento de movimiento, incluso edición de video y creación de assets para videojuegos. Los usuarios avanzados emplean la API de Python de Blender para personalizar la aplicación y escribir herramientas especializadas; a menudo estas se incluyen en futuras versiones de Blender. Blender es ideal para individuos y pequeños estudios que se benefician de su pipeline unificado y proceso de desarrollo receptivo. Ejemplos de muchos proyectos basados en Blender están disponibles en el showcase.

Blender es multiplataforma y funciona igualmente bien en computadoras Linux, Windows y Macintosh. Su interfaz usa OpenGL para proporcionar una experiencia consistente. Para confirmar compatibilidad específica, la lista de plataformas soportadas indica aquellas probadas regularmente por el equipo de desarrollo.
